Arduino Code

/* Project 24 - Ultrasonic Distance (HC-SR04)
 * Hardware: HC-SR04 with Trig->9, Echo->10. Use 5V and GND.
 * Purpose: Measure distance and react (LED/buzzer/display).
 */
const int trigPin = 9;
const int echoPin = 10;
long duration, distanceCm;

void setup() {{ Serial.begin(9600); pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); }}

void loop() {{
  digitalWrite(trigPin, LOW); delayMicroseconds(2);
  digitalWrite(trigPin, HIGH); delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  duration = pulseIn(echoPin, HIGH, 30000);  // timeout to avoid lockup
  distanceCm = (duration / 2) * 0.0343;
  Serial.print("Distance: "); Serial.print(distanceCm); Serial.println(" cm");
  // simple reaction: beep if too close
  if (distanceCm > 0 && distanceCm < 20) {{ tone(8, 1000, 100); }}
  delay(200);
}}

How It Works

Understanding the Code (line-by-line)
Key points:
1) pulseIn measures echo pulse width; convert to distance using speed of sound.
2) timeout parameter on pulseIn avoids hang if no echo.
3) check distance>0 to ignore timeout results.
4) adapt thresholds based on sensor mounting and environment.
Digital / Analog I/O & Design Notes

Digital pins set with pinMode() drive outputs (HIGH/LOW). AnalogRead reads sensors (0-1023). PWM via analogWrite uses timers to simulate analog voltages.

Expected Output

The project will perform the described behavior (LEDs/buzzer/display) when wired and uploaded correctly. Use Serial Monitor for debugging and calibration.
