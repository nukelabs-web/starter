Arduino Code

/* Project 6 - Button Counter (Serial output)
 * Hardware: Button to pin 7 (INPUT_PULLUP). Optionally display on 7-seg or LCD.
 * Purpose: Count button presses with debouncing and show how to track state changes.
 */
const int buttonPin = 7;
int count = 0;
int lastState = HIGH;
unsigned long lastDebounce = 0;
const unsigned long debounceDelay = 50;

void setup() {{ Serial.begin(9600); pinMode(buttonPin, INPUT_PULLUP); Serial.println("Ready"); }}

void loop() {{
  int reading = digitalRead(buttonPin);
  if (reading != lastState) lastDebounce = millis();
  if ((millis() - lastDebounce) > debounceDelay) {{
    if (reading == LOW && lastState == HIGH) {{ // button pressed event
      count++;
      Serial.print("Count: "); Serial.println(count);
    }}
  }}
  lastState = reading;
}}

How It Works

Understanding the Code (line-by-line)
Line-level notes:
1) Using INPUT_PULLUP simplifies wiring and removes external resistor.
2) Debounce avoids false multiple increments.
3) Serial output is a simple way to view counts; integrate with 7-seg by mapping digits to segments.
Digital / Analog I/O & Design Notes

Digital pins set with pinMode() drive outputs (HIGH/LOW). AnalogRead reads sensors (0-1023). PWM via analogWrite uses timers to simulate analog voltages.

Expected Output

The project will perform the described behavior (LEDs/buzzer/display) when wired and uploaded correctly. Use Serial Monitor for debugging and calibration.
